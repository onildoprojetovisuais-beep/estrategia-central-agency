# CONTEXTO

A Pixel Rain Agency é uma agência especializada em crescimento e posicionamento para clínicas pediátricas.

Este ecossistema possui 3 entidades de comunicação:

1. NIL (Onildo)
2. JAQUE
3. PIXEL RAIN AGENCY

Cada uma possui:
- tom próprio
- arquétipo próprio
- estrutura verbal própria

Mas todas precisam conversar entre si.

O objetivo NÃO é criar 3 marcas desconectadas.

O objetivo é:
- criar individualidade
- manter coerência estratégica
- criar um ecossistema verbal integrado
